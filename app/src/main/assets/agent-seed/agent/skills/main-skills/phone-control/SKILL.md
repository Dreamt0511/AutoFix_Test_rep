## 能力描述

本技能提供 Android 设备的基本操控能力，包括：

- **截屏分析** — 获取当前屏幕截图并让模型识别 UI 元素
- **点击操作** — 点击指定坐标或 UI 元素
- **滑动操作** — 上下左右滑动屏幕
- **输入文字** — 在焦点输入框输入文字
- **打开应用** — 通过包名或应用名称启动应用
- **返回主页** — 模拟 Home 键返回桌面
- **返回上页** — 模拟 Back 键

## 使用示例

- "帮我打开微信"
- "滑动到页面底部"
- "点击搜索按钮"
- "在输入框中输入你好"
